## Install

To install Denser Retriever, you can run:

<Tabs items={['Pip', 'Poetry']}>
  <Tab value="Pip">
  
  ```bash
  pip install git+https://github.com/denser-org/denser-retriever.git#main
  ```

  </Tab>
  <Tab value="Poetry">
  
  ```bash
  poetry add git+https://github.com/denser-org/denser-retriever.git#main
  ```
  
  </Tab>
</Tabs>

More details can be found the [DEVELOPMENT Document](https://github.com/denser-org/denser-retriever/blob/main/DEVELOPMENT.md).
